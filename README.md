#  Keen-Keeper — Keep Your Friendships Alive

### 📝 Description
**KeenKeeper** is a smart friendship management application designed to help users track and maintain their social connections. It provides a visual dashboard to monitor how long it's been since your last contact with friends, helping you stay "on-track" with your relationship goals.

---

### 🛠️ Technologies Used
- **React.js**: For building the interactive User Interface.
- **Tailwind CSS**: For modern, responsive, and pixel-perfect styling.
- **React Router DOM**: To handle seamless page navigation and dynamic routing.
- **Recharts**: For visualizing friendship analytics and interaction data.
- **React Hot Toast**: For real-time, user-friendly notifications.

---

### ✨ Key Features
1. **Dynamic Friend Dashboard:** A comprehensive list of friends with real-time status indicators (Overdue, Almost Due, On-track) that change color based on contact frequency.
2. **Interactive Timeline & Filtering:** Automatically logs every call, text, or video interaction and allows users to filter their history to see specific types of communication.
3. **Friendship Analytics:** A visual "Friendship Analytics" page featuring a Pie Chart that breaks down your communication habits, helping you see how you connect best.
